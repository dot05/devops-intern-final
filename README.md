## Overview
This project demonstrates a basic DevOps workflow using Linux, GitHub, Docker,
CI/CD with GitHub Actions, Nomad, and monitoring with Grafana Loki.

## Steps
1. GitHub repository with Python script
2. Linux shell scripting
3. Docker containerization
4. CI/CD pipeline
5. Nomad job deployment
6. Logging with Loki

## How to Run
- python hello.py
- ./scripts/sysinfo.sh
- docker build -t hello-devops .
- docker run hello-devops
- nomad job run nomad/hello.nomad
